# Code Examples for NVIDIA Isaac Integration

This directory contains code examples for Module 3: AI Perception and Navigation with NVIDIA Isaac.

## Examples Included:

1. **isaac_ros_pipeline.py** - Basic Isaac ROS pipeline setup
2. **camera_processing.py** - Camera data processing with Isaac ROS
3. **lidar_processing.py** - LiDAR data processing for navigation
4. **vslam_example.py** - Visual SLAM implementation
5. **nav2_integration.py** - Nav2 integration with Isaac ROS
6. **perception_node.py** - Custom perception node implementation

Each example demonstrates practical implementation of concepts discussed in the module chapters.
