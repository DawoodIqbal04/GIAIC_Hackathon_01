# Exercises for Navigation Algorithm Implementation

This directory contains exercises for Module 3: AI Perception and Navigation with NVIDIA Isaac.

## Exercises Included:

1. **exercise_1_path_planning.md** - Implement A* path planning algorithm
2. **exercise_2_localization.md** - Particle filter implementation for robot localization
3. **exercise_3_obstacle_avoidance.md** - Vector field histogram for obstacle avoidance
4. **exercise_4_vslam.md** - Build a simple visual SLAM system
5. **exercise_5_nav2_custom_planner.md** - Create a custom Nav2 planner plugin
6. **exercise_6_sensor_fusion.md** - Fuse IMU and odometry data using Kalman filter

Each exercise includes:
- Problem statement
- Implementation requirements
- Evaluation criteria
- Hints and resources
